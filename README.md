# NYC Airbnb Data Analysis

## Overview
This project analyzes NYC Airbnb and trip data to identify pricing trends and key factors affecting costs.

## Objectives
- Understand relationship between distance and price
- Clean and process real-world dataset
- Generate insights using data visualization

## Tools & Technologies
- Python
- Pandas
- Matplotlib

## Key Insights
- Trip distance has a positive correlation with fare amount
- Data cleaning is critical when working with real-world datasets
